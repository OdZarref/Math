Máximo Divisor Comum

O mdc(a, b) é o menor número que divide a e b.


Explicação

Para a explicação, utilizaremos 32, 48 e 64. Observe seus divisores abaixo:

Divisores do 32 são: 1, 2, 4, 8, 16, 32

Divisores do 48 são: 1, 2, 3, 4, 6, 8, 12, 16, 24, 48

Divisores do 64 são: 1, 2, 4, 8, 16, 32, 64

Podemos ver que o maior número que dividiu o 32, 48 e 64 foi o 16. Sendo assim, ele é o mdc(32, 48, 64).

Fatoração Simultânea

É possível calcular o mdc de quantos números quisermos utilizando a fatoração simultânea. Fatoramos os números simultaneamente e multiplicamos pelos fatores que dividiram toda a linha.

mdc(32, 48, 64)
 32  48  64  2 
 16  24  32  2 
 8  12  16  2 
 4  6  8  2 
 2  3  4  2 
 1  3  2  2 
 1  3  1  3 
 1  1  1  2 ** 4 = 2 x 2 x 2 x 2 = 16 

Algorítmo de Euclides

Outra forma de calcular o mdc é fazendo o seguinte processo. mdc(a, b) = mdc(a, ( | a - b | )). Como no exemplo:

A linha Q é a linha do quociente (resultado da divisão), a linha R é a linha do resto, enquanto a linha do meio é a linha que fazemos as divisões.

mdc(32, 48)
 Q  1  2 
 48  32  16 
 R  16  0 mdc(48, 32) = 16

Se quisermos, o mdc de mais de 2 números. Basta apenas calcularmos o mdc destes números de 2 em 2. Exemplo: mdc(32, 48, 64) = mdc(mdc(32, 48), 64)

Código Javascript

function mdc(a, b) {
    while (b != 0) {
        resto = a % b
        a = b
        b = resto
    }

    return a
}
