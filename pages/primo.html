Número Primo

Um número primo. É aquele que tem apenas dois divisores. O um e ele mesmo.


Explicação

Trabalhando com os números naturais, temos 2 grandes grupos. Os primos e os compostos. Primo é aquele que possui apenas dois divisores, 1 e si próprio. Sendo assim, qualquer outro número que tiver mais que 2 divisores, é denominado composto. O jeito mais fácil de saber se n é primo, é ver se ele é divisível pelos primos menores que ele. Se n for divisível por qualquer primo, então n é composto. Senão, é primo.

Exemplo: 3 é primo? O primeiro número primo e único menor que ele é o 2. Então 3/2 é igual a 1 e resto 1. Logo, 3 não é divisível pelos primos menores que ele. Então ele é primo!

Código Javascript

function calcular_primo(possivel_primo) {
    divisores = 0
    numero = 0

    while (possivel_primo != numero) {
        numero ++

        if (possivel_primo % numero == 0) {
            divisores ++
        }
    }

    if (divisores == 2) {
        return true
    } else {
        return false
    }
}
