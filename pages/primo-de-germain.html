Primo de Germain

Onde p é primo, se 2p + 1 for também primo. Então p é primo de Sophie Germain.




Explicação

A matemática francesa Marie-Sophie Germain, definiu um novo tipo de número. Que tendo um número primo p, se 2p + 1 também for primo, então p é primo de Germain.

Exemplo com o primo 3:

2 x p + 1
2 x 3 + 1
6 + 1
7

O que resulta no 7, que é um número primo, o que faz do 3 um primo de Germain.

Exemplo com o primo 11:

2 x p + 1
2 x 11 + 1
22 + 1
23

E 23 também é primo. Logo 11 é primo de Germain.

Código Javascript

//Vê se "possivel_primo" é de fato primo
function calcular_primo(possivel_primo) {
    divisores = 0
    numero = 0

    while (possivel_primo != numero) {
        numero ++

        if (possivel_primo % numero == 0) {
            divisores ++
        }
    }

    if (divisores == 2) {
        return true
    } else {
        return false
    }
}

//Vê se "numero" é primo de Germain
function primo_germain(numero) {
    if (calcular_primo(numero * 2 + 1)) {
        return true
    } else {
        return false
    }
}
