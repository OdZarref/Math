Fatorial

Dado como n!. É o produto das multiplicações de n pelos números naturais menores que ele (com excessão do zero).


O cálculo do fatorial de n, consiste na multiplicação sucessiva de n pelos números naturais menores que ele.

Exemplo: 5! = 5 . (5-1) . (5-2) . (5-3) . (5-4) = 5 . 4 . 3 . 2 . 1 = 120

Código Javascript

function fatorial(n) {
    if (n == 0 || n == 1) {
        return 1
    } else {
        return n * fatorial(n-1)
    }
}
