Sobre

Math! é um projeto de site Open-Source de cálculos. Feito com o intuito de unir HTML, CSS, programação e matemática. É uma boa base de projeto para qualquer iniciante. Acesse o código-fonte e inspire-se ou contribua para o projeto!

Você pode fazer uma série de cálculos e ver as respectivas funções Javascript que foram utilizadas para realizá-los. Muito mais que um site de cálculos, o principal objetivo é mostrar para aqueles que gostam de matemática e principalmente para aqueles que gostam de programação, como é feito tal cálculo.

Pequeno projeto pessoal, codado completamente por uma única pessoa e hospedado através do Github Pages. Todo o código-fonte está disponível para consulta de qualquer interessado.

Acesse!
