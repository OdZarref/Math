Decomposição em Fatores Primos

Os fatores de n se dão através da divisão sucetiva de n por números primos. Os primos que forem divisores de n, são seus fatores.


Explicação

Todo número composto, ou seja, todo número que tem mais de 2 divisores, pode ser escrito através de seus fatores primos. Que são encontrados através da decomposição deste por números primos. Observe:

362
182
93
33
12 x 2 x 3 x 3 = 36

Com isso encontramos os fatores primos de 36, que são 2 ^ 2 x 3 ^ 2.

Código Javascript


//Calcula o número primo
function primo(possivel_primo) {
    divisores = 0
    numero = 0

    while (possivel_primo != numero) {
        numero ++

        if (possivel_primo % numero == 0) {
            divisores ++
        }
    }

    if (divisores == 2) {
        return true
    } else {
        return false
    }
}

//Calcula os fatores primos
function fatores_numero(n) {
    fatores = []
    primo = 2

    while (n != 1) {
        if (n % primo == 0 && primo(primo)) {
            fatores.push(primo)
            n /= primo
        } else {
            if (primo == 2) {
                primo ++
            } else {
                primo += 2
            }
        }
    }

    return fatores
}
