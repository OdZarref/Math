Mínimo Múltiplo Comum

Tendo como base a e b, o mmc(a, b) é o menor múltiplo simultâneo de a e b.


Explicação

Observe as tabuadas abaixo:

3 x 1 = 36 x 1 = 610 x 1 = 10
3 x 2 = 66 x 2 = 1210 x 2 = 20
3 x 3 = 96 x 3 = 1810 x 3 = 30
3 x 4 = 126 x 4 = 2410 x 4 = 40
3 x 5 = 156 x 5 = 3010 x 5 = 50
3 x 6 = 186 x 6 = 3610 x 6 = 60
3 x 7 = 216 x 7 = 4210 x 7 = 70
3 x 8 = 246 x 8 = 4810 x 8 = 80
3 x 9 = 276 x 9 = 5410 x 9 = 90
3 x 10 = 306 x 10 = 6010 x 10 = 100

Note que 30 é o menor produto que aparece nas multiplicações de 3, 6 e 10. Logo, o mmc(3, 6, 10) é o 30!

Para explicação o exemplo acima serve muito bem, mas como notamos, para calcular números maiores, é extremamente massante. Para isso existe um método de encontrar o mmc de dois ou mais números através da multiplicação dos fatores primos dos mesmos. Observe abaixo:

mmc(3, 6, 10)
 3  6  10  2
 3  3  5  3
 1  1  5  5
 1  1  1  2 x 3 x 5 = 30

Fato é que existe uma relação entre o mmc e mdc. Sendo assim é possível calcular o mmc através do mdc. Para isso utilizei a seguinte função:

mmc(a, b) = a x (b / mdc(a, b))

Se quisermos calcular o mmc de mais de 2 números. Para o mmc(a, b, c). Calculamos primeiro o mmc(a, b), e depois calculamos o mmc do resultado com c. Exemplo:

mmc(a, b, c) = mmc(mmc(a, b), c)

Código Javascript

// Calcula o mdc
function mdc(a, b) {
    while (b != 0) {
        resto = a % b
        a = b
        b = resto
    }

    return a
}

// Calcula o mmc com base no mdc
function mmc(a, b) {
    return a * (b / mdc(a, b))
}
